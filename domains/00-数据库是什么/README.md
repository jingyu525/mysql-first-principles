# 领域视角：数据库是什么

> 先不学任何语法，先回答一个根本问题：为什么会有数据库这个物种？

---

## 核心问题

| 问题 | 答案 |
|------|------|
| 为什么需要数据库？ | 程序重启 → 内存数据消失 → 需要持久化到磁盘 |
| 如何快速找到数据？ | 1000 万条记录不能线性扫描 → 需要索引 |
| 多人同时修改怎么办？ | 余额扣款冲突 → 需要事务 |
| 数据库的本质是什么？ | **管理磁盘数据的操作系统** |

## 关键认知

1. **数据库不是 SQL 语言** — SQL 只是入口，真正在做的是磁盘数据管理
2. **MySQL ≠ InnoDB** — MySQL Server 负责 SQL 解析优化，InnoDB 负责存储
3. **一切设计围绕磁盘 IO** — 数据库性能瓶颈不在 CPU，而在磁盘（慢 10 万倍）
4. **Page 是核心单位** — 数据库以 Page（16KB）为单位工作，不是以行为单位
